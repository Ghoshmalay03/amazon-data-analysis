# Amazon Product Data Analysis

## Project Overview
This project analyzes Amazon product data using Python.

## Features
- Data Cleaning
- Exploratory Data Analysis
- Visualizations
- Business Insights

## Libraries Used
- Pandas
- Matplotlib
- Seaborn

## Dataset
Amazon Product Dataset from Kaggle.

## Key Insights
- Electronics dominate listings
- Most products are highly rated
- Discounts are widely used
